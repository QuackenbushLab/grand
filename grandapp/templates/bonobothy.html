BONOBO-PANDA networks are single-sample networks generated by first estimating single-sample co-expression networks for each sample and then running PANDA on each single-sample co-expression with motif and PPI data. The table below has a description of the primary clinical variables of the gene expression samples that were used to generate the gene regulatory network. It is possible to download each network individually by selecting the sample network to download. The network file has a TF-by-gene (997x26469) matrix named after the GTEx sample reference.
